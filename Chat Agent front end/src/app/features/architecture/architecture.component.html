Key Design Principles

auto_awesome

Fully Serverless

No EC2, no ECS, no containers. Lambda + DynamoDB + S3 = zero server management.

 trending_up

Auto-Scaling

Every service scales automatically. DynamoDB on-demand, Lambda concurrency, CloudFront edge caching.

 code

Infrastructure as Code

Entire stack in a single SAM template.yaml — reproducible, auditable, version-controlled.

 security

Security by Default

Cognito auth, RBAC, tenant isolation, encrypted at rest and in transit, AI guardrails.

Request Lifecycle

Admin Configurator (Authenticated)

1
Angular SPA User signs in via Cognito, receives JWT tokens
2
API Gateway Validates JWT, extracts user identity, routes to Lambda
3
Lambda (API) Checks RBAC role, processes request, queries DynamoDB
4
DynamoDB Returns tenant-scoped data (filtered by organizationId)

Chat Widget (Public, API Key Auth)

1
Embedded Widget JS loaded on host page, sends API key + message
2
API Gateway No JWT required — widget endpoints bypass Cognito auth
3
Lambda (API) Validates API key, looks up assistant config, invokes Bedrock Agent
4
Bedrock Agent Retrieves from Knowledge Base, generates response via Claude model
5
Response Streamed back through API Gateway to widget in real-time

Scheduled Report Delivery

1
EventBridge Scheduler Fires cron trigger at scheduled time (timezone-aware)
2
Provision Lambda Reads schedule from DynamoDB, creates run record
3
Snowflake Lambda Executes SQL, generates Excel/CSV, uploads to S3
4
SES / Twilio Delivers secure download link via email/SMS (no PHI in message body)
5
Recipient Clicks link → reports.wubba.ai CDN → S3 download (7-day retention)

Screen Context (Widget URL Matching)

1
Widget Init On page load, widget sends current URL + API key
2
Lambda Matches URL against screen mapping regex patterns
3
Response Returns contextual videos, help articles, and trending questions
