  1. Copy the HTML snippet above and paste it before the closing </body> tag on your website.
  2. Set allowed domains on your assistant to restrict which sites can load the widget. Leave empty to allow any domain.
  3. Optional: Use a node-scoped snippet if you want different hierarchy nodes (e.g. offices or companies) to have independent API keys and analytics.
